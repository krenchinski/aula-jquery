Perguntas Frequentes

O que é JQuery? +
É uma biblioteca JavaScript rápida, pequena e rica em recursos, criada para facilitar o uso do JavaScript no desenvolvimento de páginas web.
Por que usar JQuery? +
Facilita o uso do JavaScript, escreve menos o código e funciona em todos os navegadores
Como incluir o JQuery em uma página? +
Via CDN ou baixando o arquivo jquery
Como incluir o JQuery em uma página? +
Adicione este código dentro do seu HTML, normalmente no head ou antes do fechamento do /body
Quais os tipos de seletores do jQuery para capturar ID e Class? +
Usando seletores CSS muito semelhantes aos usados em CSS puro
Porque devemos capturar eventos dentro de um $(document).ready({ /* código */})? +
Porque, se você tentar manipular elementos que ainda não foram carregados, o jQuery não vai encontrá-los, seu código pode quebrar ou simplesmente não funcionar
Qual a diferença do evento de keyup e keydown? +
Os eventos keydown e keyup são usados para detectar quando uma tecla do teclado é pressionada, mas eles ocorrem em momentos diferentes do processo de pressionar uma tecla.
Qual a diferença do jQuery para o JavaScript puro? +
A diferença está principalmente em como você escreve o código, e não no que você pode fazer, porque jQuery é uma biblioteca feita com JavaScript.
